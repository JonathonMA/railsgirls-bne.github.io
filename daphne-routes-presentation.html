Routes

A system called routing in Rails, matches HTTP requests from browser to the app

HTTP methods

Requests made from browser (by typing in a URL) would have a HTTP method defined so that the app knows what to document

HTTP methods

There are 4 HTTP methods:

  • GET
  • PUT
  • POST
  • DELETE
